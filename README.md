## 📒 Getaway with the Cash, [ 돈가방을 갖고 튀어라 플러그인 ]
> 마인크래프트 서버 플러그인 (1.12.2 버전)
  
---  

#### 📖 어떤 플러그인인가요?
마인크래프트에서 **돈가방을 갖고 튀어라** 콘텐츠를 구현한 플러그인입니다.

#### 📄 명령어
```  
* /가방 <탐지/진짜/일반> - 가방을 생성합니다.(열려면 열쇠 필요, 버려도 사라지지 않음)
* /열쇠 [강제] - 손에 든 가방에 대한 열쇠를 생성합니다.
* /좌표 - 모든 가방에 대한 좌표를 출력합니다.
```
